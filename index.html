unique features

Stellar Inu is a utility token built on the Ethereum Mainnet, which means it can be used to access specific services or products within the project's ecosystem.

There is a 5% tax rate on all transactions made with Stellar Inu. This tax is used to fund buy backs, burns, marketing, development, and community giveaways.

The project does not have any tokens reserved for the team, which suggests that the team is focused on the project's long-term success rather than short-term gains.
